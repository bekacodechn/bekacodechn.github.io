#How accurate is this?

This method is not 100% accurate, but it's as close as web developers can get with currently available browser APIs.

A few points of note:

  • requestAnimationFrameÂ won't run if the page is hidden (i.e. in a background tab), so you should exclude hidden tabs from your measurements.
  • MessageChannelÂ onmessageÂ callbacks are high priority, but input events likeÂ clickÂ andÂ keypressÂ take precedence, and so it won't always run immediately after a Frame is produced.
